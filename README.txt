To test, create a new directory outside the source directory
1. mkdir CMakeTutorial-build
2. cd CMakeTutorial-build
3. ccmake <path to CMakeTutorial-build>

NOTE:
1. These are just a few steps of the official tutorial at 
http://www.cmake.org/cmake/help/cmake_tutorial.html

2. Detailed explanations have been added where necessary

3. You can fork and extend and test whatever you want
   All source files for the official tutorial for a step-by-step understanding
   can be downloaded from
   http://public.kitware.com/cgi-bin/viewcvs.cgi/Tests/Tutorial/?root=CMake
